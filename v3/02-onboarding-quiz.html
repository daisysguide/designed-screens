Question count is dynamic: 6 questions (Thinking about trying) · 7 questions (Expecting again — Q3 auto-set) · 8 questions (Currently pregnant / Baby is already here).

Conditional routing: Q2 (due date picker) only shown when Q1 = "Currently pregnant" OR "Expecting again" · Q2b (baby's birthday picker) shown when Q1 = "Baby is already here" · Q3 skipped entirely when Q1 = "Expecting again" — auto-set to "No" in data layer · Q2 and Q3 both skipped when Q1 = "Thinking about trying"

Back chevron returns to previous question in sequence; from Q1, returns to Screen 06 (Onboarding Intro / Swipeable Intro). Completing all questions → Screen 03. Quiz state is persisted to localStorage after each answer and cleared on completion or back past Q1.
All questions + conditional logic
Q1
Where are you in your journey?
Options: Thinking about trying · Currently pregnant · Baby is already here · Expecting again · Required — no skip · Determines entire question sequence
Q2
When is your due date?
Native date picker · Min date: today · Label: "Due date" · Shown when Q1 = "Currently pregnant" OR "Expecting again" · Required — no skip
Q2b
When was baby born?
Native date picker · Max date: today · Label: "Baby's birthday" · Shown when Q1 = "Baby is already here" · Used for age-appropriate topic sequencing · Required — no skip
Q3
Is this your first baby?
Options: Yes, first time · No, we've done this before · Shown when Q1 = "Currently pregnant" or "Baby is already here" · Skipped (auto-set "No") when Q1 = "Expecting again" · Skipped entirely when Q1 = "Thinking about trying" · Required when shown — no skip
Q4
Have you found out the baby's sex yet?
Options: Yes, it's a girl · Yes, it's a boy · Not yet · We're not finding out · Filters circumcision topic (excluded only for "Yes, it's a girl") · Optional · Skip = treat as "We're not finding out" (circumcision topic included)
Q5
Will you be sharing a last name?
Options: Yes · No · Still deciding · Filters last name topic · Optional · Skip = treat as "Still deciding" (last name topic included)
Q6
Are cultural or religious traditions important to you?
Options: Yes, very · Somewhat · Not really · Optional · Skip = treat as "Somewhat"
Q7
How often would you like us to nudge you?
Options: Daily · A few times a week · Weekly · Don't nudge me · Optional · Skip = "A few times a week" · Skip label: "Skip — I'll set this in Settings later"
Q8
How do you want to work through your topics?
Options: Guided timeline · Self-directed · Optional · Skip = "Guided timeline" · Skip label: "Skip — I'll set this in Settings later"

Behavior notes
One question per screen — same template reused for all questions in the sequence.
Q1 answer immediately determines the full sequence and updates the progress bar total. Sequence and progress label recalculate before advancing to Q2/Q2b.
"Expecting again" silently sets Q3 = "No" in the data layer without showing the question. The user's second-time parent status is inferred.
Q4 circumcision filtering: excluded only when answer is "Yes, it's a girl". All other answers (Yes, it's a boy / Not yet / We're not finding out / skipped) keep the topic included.
Saved state is restored on relaunch — previous answers are pre-selected. User resumes at the first unanswered question.
Tapping a selected option deselects it — Continue returns to disabled state.
Back chevron returns to previous question in the current sequence. From Q1, returns to Screen 01 and clears saved state.
No bottom nav on this screen.
Quiz answers are persisted to local storage after each question. On relaunch mid-quiz, the user resumes from their last unanswered question with all prior answers pre-populated. State clears on completing the quiz or navigating back past Q1.
When Q1 = "Expecting again": Q2 (due date) is shown, Q3 is skipped and auto-set to "No" in the data layer. User proceeds from Q2 to Q4.
When Q1 = "Baby is already here": Q2b (baby's birthday) is shown instead of Q2. Q3 is shown normally.
When Q1 = "Thinking about trying": Q2 and Q3 are both skipped. User proceeds from Q1 to Q4.
