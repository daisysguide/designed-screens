Behavior notes
Each digit auto-advances focus to the next box. Backspace moves focus back.
iOS SMS autofill pastes all 6 digits at once — handle as a single paste event.
"Resend code" is disabled for 30 seconds after the screen loads. Countdown pill shows remaining time, disappears at 0.
Wrong code: show error state on all boxes + error message. Don't clear digits. User corrects in place.
Back button and "Go back" both return to Screen 02.
No bottom nav on this screen.
